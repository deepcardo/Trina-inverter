# Excel 文件比对工具

## 功能

比对两个 Excel 文件中「区域」维度的并网箱、逆变器数据差异，输出变更明细和最新数据。

## 文件说明

| 文件 | 说明 |
|------|------|
| `启动比对工具.bat` | 一键启动（双击即可） |
| `compare_gui.py` | 图形界面程序 |
| `compare.py` | 核心比对引擎（被 GUI 调用） |
| `全国容配比查询表0407.xlsx` | 旧版数据文件（示例） |
| `全国容配比查询表0506.xlsx` | 新版数据文件（示例） |

## 使用方法

### 方式一：双击启动（推荐）

双击 `启动比对工具.bat`，等待界面打开。

### 方式二：命令行启动

```bash
python compare_gui.py
```

## 操作步骤

1. **选择原文件** — 点击「选择文件」或按 `Ctrl+O`，选取旧版 Excel
2. **选择比对文件** — 点击「选择文件」或按 `Ctrl+N`，选取新版 Excel
3. **选择输出目录**（可选）— 默认输出到工具所在目录
4. 点击 **「开始比对」** 或按 `Enter`

## 输出结果

比对完成后在输出目录生成 `变更记录.xlsx`，包含两个工作表：

- **变更明细** — 逐条列出旧值和新值，新值用红色标记
- **最新数据** — 只展示每条区域的最新值，不显示旧值

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+O` | 选择原文件 |
| `Ctrl+N` | 选择比对文件 |
| `Enter` | 开始比对 |

## 注意事项

- 两个 Excel 文件需包含「区域」「省份」「市」「区县」「并网箱」「逆变器」列
- 依赖 `pandas` 和 `openpyxl`，启动脚本会自动安装缺少的依赖
- 需要 Python 3.8+
