# start-ui 使用说明

## 概述

`start-ui.bat` 是一个一键启动工具，用于在本机运行 Excel 转换 / 测试 UI 的 Web 界面。

## 使用方法

直接双击 `start-ui.bat` 即可运行，或从命令行执行：

```batch
start-ui.bat
```

## 功能菜单

| 选项 | 功能 | 说明 |
|------|------|------|
| 1 | 数据转换 | 打开 convert-excel-ui 页面 |
| 2 | 测试运行 | 打开 test-verify-ui 页面 |
| 3 | 同时打开 | 同时打开以上两个页面 |
| 0 | 退出 | 直接退出程序 |

## 前置条件

- **Python 3.x** — 用于启动内置 HTTP 服务器
- 端口 **8080** 未被占用

## 运行流程

1. 检测 Python 环境
2. 检查 8080 端口是否可用
3. 显示功能菜单供选择
4. 启动 Python HTTP 服务器（`http.server` 模块）
5. 在默认浏览器中打开对应的 HTML 页面
6. 按任意键停止服务器并退出

## 项目结构

```
D:\Code\tools\
├── scripts\
│   ├── start-ui.bat          ← 启动脚本（本文件）
│   ├── convert-excel-ui.html ← 数据转换页面
│   └── test-verify-ui.html   ← 测试运行页面
└── ...其他工具文件
```

## 注意事项

- 服务器运行期间请勿关闭命令行窗口
- 按任意键即可安全停止服务器
- 如果端口被占用，脚本会给出提示并退出
